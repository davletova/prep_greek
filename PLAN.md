# План этапа 1 — статический Telegram Mini App (hello world)

## Цель
Минимальный Mini App, который открывается в Telegram, читает JSON из репозитория и показывает строку "hello, world!".

## Шаги
1. Подтвердить структуру и место контента: `webapp/public/content/hello.json`.
2. Создать минимальный Vite + React webapp, который читает `/content/hello.json` и отображает сообщение.
3. Добавить инициализацию Telegram WebApp (`Telegram.WebApp.ready()`), чтобы приложение корректно открывалось внутри Telegram.
4. Настроить GitHub Pages для webapp (сборка и публикация статических файлов).
5. Проверить end-to-end: открыть Mini App по GitHub Pages URL в Telegram и увидеть "hello, world!".

# План этапа 2 — экран алфавита

## Цель
Добавить стартовый экран и экран изучения алфавита с навигацией и озвучкой.

## Шаги
1. Определить схему данных и создать `webapp/public/content/theory/alphabet.json` с буквами и данными для озвучки.
2. Реализовать стартовый экран по референсам с кнопкой “Изучить алфавит”.
3. Реализовать экран алфавита (4 плашки на экране, стиль по референсам), открываемый по кнопке “Изучить алфавит”.
4. Добавить кнопку “выйти” (возврат на стартовый экран) и навигацию “вперед/назад” по всему алфавиту.
5. Добавить озвучку букв через Web Speech API.
