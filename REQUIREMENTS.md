# Требования к проекту

Цель: подготовка к экзамену по греческому языку (ΚΕΓ), уровни A2 и B1.

## Форматы тренировок (MVP)

Чтение:
- True/False по тексту.
- Matching: сопоставить начало и конец фразы.
- Несколько коротких текстов + выбор правильного утверждения (multiple choice).
- Cloze: пропуски + банк слов.

Аудирование:
- Прослушай и соотнеси с утверждением/предложением/фразой.
- Multiple choice по диалогу.
- True/False по аудио.
- Заполнение формы по аудио (note-taking).

Письмо:
- Письмо/сообщение другу (80–100 слов) по сценарию.
- Объявление/анонс (80–100 слов).

## Генерация контента (AI)

- Генерация по запросу тренировки: один запрос на пакет заданий (например, 10 однотипных задач).
- Автопроверка заданий выполняется AI.
- Письменные задания проверяются AI.

## Оценивание и прогресс

- В MVP оценивание не требуется.
- Далее: результат после каждой тренировки (пример: 10 однотипных заданий → итоговый результат).
- Статистика пользователя пока не хранится.

## Аудирование

- Используются стандартные бесплатные средства аудирования (без расширенных настроек).

## Ограничения

- Лимиты (длительность сессий/количество заданий) будут определены позже.
